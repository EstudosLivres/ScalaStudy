#### Variable

- Constant 
    - initial const (__val__ reserved word)
        ```Scala
          val name = "King"
        ```
    - late const (__val__ reserved word)
        ```Scala
          def name = "King"
        ```
    
- Variable (__var__ reserved word)
    ```Scala
      var name = "King"
    ```
